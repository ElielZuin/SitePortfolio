DropSense — Processamento de Imagens Hidrossensíveis Offline

Análise confiável, performance local e ativação segura. Tecnologia projetada para precisão e privacidade em campo.

Sobre o Projeto

O DropSense é um sistema de análise de cartões hidrossensíveis desenvolvido para funcionar totalmente offline. Seu propósito é simplificar o processo de análise em campo, permitindo que o usuário capture e processe imagens localmente, sem necessidade de servidores externos. A única conexão exigida é durante a ativação inicial, garantindo assim uma camada de segurança e controle de licenças.

Arquitetura Técnica

O sistema é composto por dois componentes principais:

  • App Android (Java) — Responsável por toda a análise de imagem, interface e execução dos cálculos de métricas, como densidade de manchas, distribuição e taxa de cobertura. Utiliza OpenCV para processamento e roda integralmente no dispositivo.
  • API de Ativação (.NET / ASP.NET Core) — Gerencia o licenciamento do aplicativo. Cada chave é autenticada via criptografia assimétrica e validada diretamente no dispositivo, sem necessidade de conexão contínua.

A arquitetura foi projetada para ser modular, permitindo futuras expansões como integração com relatórios e sincronização de resultados, mantendo o núcleo local por questões de privacidade e latência zero.

Segurança e Logs Estruturados

A segurança foi um pilar central do projeto. Implementamos camadas de proteção contra engenharia reversa, ofuscação do código Java e verificação de integridade no momento da execução. A comunicação com a API ocorre apenas em HTTPS com pinning de certificado.

Os logs estruturados são utilizados para auditoria e suporte técnico, organizados em níveis de severidade e contexto. Nenhum dado sensível é registrado, apenas metadados de operação e estado, garantindo rastreabilidade sem comprometer a privacidade do usuário.

Processamento Local e Performance

O processamento das imagens é feito integralmente no dispositivo, com algoritmos otimizados em Java. Foram implementadas rotinas matemáticas específicas para cálculo de área de deposição e densidade média visual (DMV). Versões iniciais em C# foram utilizadas como base para validação dos modelos, sendo posteriormente adaptadas e refinadas para Java.

Durante o desenvolvimento, foram identificadas inconsistências em alguns cálculos de distribuição e calibragem de contraste, que foram ajustadas com base em resultados experimentais reais. O resultado final é um algoritmo robusto, leve e preciso.

Resultados e Conclusão

Com o DropSense, foi possível atingir um tempo médio de análise inferior a 2 segundos por imagem em dispositivos intermediários, com consistência nos resultados entre diferentes modelos de smartphones. A arquitetura híbrida — local e com ativação via API — garante segurança, privacidade e flexibilidade, sem depender de conectividade constante.

Este projeto reflete um equilíbrio entre engenharia de software, segurança da informação e visão computacional, mostrando a importância de projetar soluções performáticas e seguras mesmo em ambientes offline.
